# Backend projeto final de PDI

## Instalação

### Utilizando venv

Crie um venv (recomendado)

```bash
python -m venv .venv
```

Ative o venv

```bash
source .venv/bin/activate
```
### Instalação das dependências

```bash
pip install -r requirements.txt
```

### Rodar o servidor local 

```bash
cd app
make run-dev
```
